# MaGIC

Sito web di MaGIC.

## Per testare localmente

- Installare Ruby
- Installare Bundler
- Scendere nel repo
- Opzionale: `bundle config set path vendor` (installa i package
  localmente piuttosto che nei cammini di sistema)
- `bundle install` per installare le dependencies
- `bundle exec jekyll serve` per testare il sito localmente
